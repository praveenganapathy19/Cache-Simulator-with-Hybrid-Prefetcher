Spring 2017

Project--Implementation of a 2-level Cache Simulator with a Hybrid Prefetcher

Team Members: Praveen Ganapathy Narasimhan, Supraja Mahadevan, Sowmya Suresh Chander, Shanmathi Rajasekar



To build the cache simulator, use the following command:



	make



To clean up files, you may use:



	make clean



To run the file on Windows Command Prompt, use the following command:



	cacheSim.exe g++.trace



Results are stored in g++.trace.out
